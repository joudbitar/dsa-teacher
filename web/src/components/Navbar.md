# Navbar Component

## Purpose

Top navigation bar for the application. Displays branding, navigation links, and user authentication status.

## Props

```typescript
// No props; uses global auth context
```

## Interactions

- Logo/brand links to landing page
- "Challenges" link navigates to challenges list
- "My Projects" link navigates to user's project dashboard
- Auth button: "Sign In" (logged out) or user avatar menu (logged in)

## Visual Notes

- Sticky positioning at top
- Dark background with accent highlights
- Responsive: hamburger menu on mobile
