# OnboardingSteps Component

## Purpose

Illustrates the product loop: pick challenge → clone repo → test → submit → see results.

## Props

```typescript
// No props; static steps
```

## Interactions

- Purely informational; no clickable actions

## Visual Notes

- Horizontal timeline (desktop) or vertical list (mobile)
- Five steps:
  1. Pick a Challenge
  2. Get Your Private Repo
  3. Code Locally & Test
  4. Submit Your Solution
  5. Track Progress
- Icons/illustrations for each step
- Connected by dotted line or arrow
